Diamond Line

Project Date Feb 1st, 2025
Tags Mobile App Developer, Flutter , Peaklink
Client Diamond Line - Syria

In This Project My Responsibility Was:

- Fully Design And Logic Work

- Clean Code

- RESTful APIs

- Getx State Management

- Translation (EN-AR)

- RTL Designs

- Google Maps With Custom Markers with realtime tracking for drivers

- Socket.io Integration

Diamond Line is a ride-hailing application similar to Uber, featuring both user and driver apps. The project includes real-time tracking capabilities, allowing users to track their drivers in real-time on the map. Developed as part of the Peaklink team, this comprehensive solution demonstrates advanced mobile app development skills using Flutter.
